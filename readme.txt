Share your contents with our share button in Tubaret Network, a place where you can write and share news and articles about your favourite topics. 

== Description ==

The Tubaret Network Share Plugin for Wordpress puts a little button in all your articles to share your contents in the Tubaret Network.

The Tubaret Network is composed by:

http://www.tubaret.com 

http://www.gameret.com

http://www.indieret.com

http://www.viajeret.com

== Installation ==

Just access to the Plugins control panel, search for Tubaret Network share plugin and enable it.

You can config some parameters like the target tubaret network site, style and position in Admin Panel -> Settings -> Tubaret Network.
